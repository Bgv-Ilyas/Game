# RusRunner (Diploma Project)

Мультиплатформенный (телефоны + ноутбуки) 2D автораннер на Unity/C# в пиксель-арт сеттинге Древней Руси.

## Общая тема диплома
- Проектирование, оптимизация и реализация программного комплекса казуальной 2D-игры на основе жанра линейного автораннера.

## Личная тема диплома
- Проектирование и оптимизация модульного ядра для 2D-автораннера.

## Цели проекта
- Реализовать расширяемое модульное ядро игры.
- Снизить стоимость добавления нового контента (способности, усиления, препятствия, сегменты уровня).
- Обеспечить стабильную работу на мобильных устройствах и ноутбуках.
- Подготовить основу для модернизации и дальнейшего развития.

## Технологический стек
- Unity 2D (URP optional)
- C#
- ScriptableObject для конфигов
- Addressables (на следующем этапе)

## Базовая структура (MVP)
- `Assets/Scripts/Core` — ядро, сервисы, события.
- `Assets/Scripts/Game` — логика автораннера.
- `Assets/Scripts/Content` — способности, усиления, конфиги.
- `docs` — архитектура, план диплома, оптимизация, тестирование.

## Идея геймплея
- Игрок управляет богатырем/гонцом, который бежит по линейному маршруту.
- Джамп/слайд/рывок + временные усиления.
- Случайно составляемые сегменты трассы (с контролем сложности).
- Пиксельная графика в стилистике Древней Руси (избы, частоколы, ярмарка, лес, стража, тотемы и т.д.).
